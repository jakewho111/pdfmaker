Frequently Asked Questions

Why is the compressed file the same size sometimes? +
PDF compression in the browser works by optimizing the PDF structure and object streams. Text-only PDFs are already very compact so the difference may be minimal. PDFs with lots of embedded images compress much more significantly (30–70%). For maximum compression of image-heavy PDFs, use the High setting.
What does each compression level do? +
Low uses basic object stream optimization. Medium adds additional cross-reference stream compression. High applies maximum object stream packing and removes metadata to squeeze the most bytes possible out of the file structure.
Is my file safe? +
Completely. Everything runs in your browser. Your file never leaves your device.
